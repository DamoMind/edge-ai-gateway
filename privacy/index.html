ProductInsight 隐私政策

最后更新：2025年1月11日

概要：ProductInsight 尊重您的隐私。我们不收集个人身份信息，不存储您浏览的商品数据，仅收集匿名使用统计以改进产品。

1. 我们收集的信息

ProductInsight 收集以下匿名数据用于改进产品体验：

  • 使用统计：扩展的安装、更新、页面浏览次数
  • 功能使用：AI 分析次数、刷新操作（不含商品内容）
  • 错误信息：匿名错误报告，用于修复问题

2. 我们不收集的信息

  • 个人身份信息（姓名、邮箱、地址等）
  • 您的 Amazon 账户信息
  • 您浏览的具体商品详情
  • 您的购买历史或购物车内容
  • 您的浏览历史

3. 数据处理方式

当您使用 AI 分析功能时：

  • 商品信息（标题、价格、规格等）会发送到我们的 AI 服务进行分析
  • 分析完成后，数据不会被存储
  • 我们不会将您的数据出售或分享给第三方

4. 数据存储

  • 本地存储：使用偏好（语言设置、每日使用次数）存储在您的浏览器本地
  • 无服务器存储：我们不在服务器上存储任何用户数据

5. 第三方服务

我们使用以下第三方服务：

  • Google Analytics：收集匿名使用统计
  • Azure OpenAI：提供 AI 分析能力（通过安全代理）

6. 数据安全

  • 所有数据传输使用 HTTPS 加密
  • AI API 密钥安全存储在服务器端，不在扩展代码中暴露

7. 您的权利

  • 您可以随时卸载扩展以停止所有数据收集
  • 本地存储的数据会随扩展卸载自动清除

8. 政策更新

我们可能会不时更新此隐私政策。更新后的政策将在此页面发布，并更新"最后更新"日期。

9. 联系我们

如果您对此隐私政策有任何疑问，请通过以下方式联系我们：

  • GitHub Issues: 提交问题

ProductInsight Privacy Policy

Last Updated: January 11, 2025

Summary: ProductInsight respects your privacy. We do not collect personal information, do not store your browsing data, and only collect anonymous usage statistics to improve our product.

1. Information We Collect

ProductInsight collects the following anonymous data to improve user experience:

  • Usage Statistics: Extension installs, updates, page views
  • Feature Usage: AI analysis count, refresh actions (without product content)
  • Error Information: Anonymous error reports for bug fixes

2. Information We Do NOT Collect

  • Personal identification information (name, email, address, etc.)
  • Your Amazon account information
  • Specific product details you browse
  • Your purchase history or shopping cart contents
  • Your browsing history

3. How We Process Data

When you use the AI analysis feature:

  • Product information (title, price, specs) is sent to our AI service for analysis
  • Data is not stored after analysis is complete
  • We do not sell or share your data with third parties

4. Data Storage

  • Local Storage: User preferences (language settings, daily usage count) are stored locally in your browser
  • No Server Storage: We do not store any user data on our servers

5. Third-Party Services

We use the following third-party services:

  • Google Analytics: For anonymous usage statistics
  • Azure OpenAI: For AI analysis capabilities (via secure proxy)

6. Data Security

  • All data transmission is encrypted using HTTPS
  • AI API keys are securely stored server-side, not exposed in extension code

7. Your Rights

  • You can uninstall the extension at any time to stop all data collection
  • Locally stored data is automatically cleared when the extension is uninstalled

8. Policy Updates

We may update this privacy policy from time to time. Updated policies will be posted on this page with a new "Last Updated" date.

9. Contact Us

If you have any questions about this privacy policy, please contact us via:

  • GitHub Issues: Submit an Issue
